Agenda

Agenda para controle do projeto

Fase 1:

11/02/2026

O grupo criou uma agenda para facilitar no controle do projeto, onde os integrantes fazem anotações sobre e alterações se necessário, sempre marcando data, nome, horário.

12/02/2026

O grupo começou a desenvolver o código do projeto com base no código que um dos integrantes já havia feito, tendo feito html, css e js.

25/02/2026

Um dos integrantes se reuniu com a cliente do projeto e ambas discutiram sobre do que precisa, do design, a interface e sobre ideias de como poderia ficar.

27/02/2026

Hoje, um dos integrantes pesquisou sobre criar um banco de dados para armazenar as informações do sistema de cadastro, facilitando na hora de gerenciar, modificar, e apagar (se necessário).

05/03/2026

Hoje o grupo foi até a professora do curso pedir informações sobre criação de banco de dados, organização de equipe, opiniões e comentarios pessoais dela sobre o TCC.

11/03/2026

Um dos integrantes do grupo fez um novo rascunho do projeto e esta sendo testado para saber se vai dar certo. E realmente deu. Ficou com uma visibilidade melhor que a outra, dando realmmente um ar de sistema de cadastro hospitalar. A cliente esta adorando o caminhar do projeto e esta muito animada com o resultado final dele.

18/03/2026

A professora ensinou a como fazer um fork no github para que o desenvolvimento seja mais fácil e rápido. Com isso, um dos integrantes subiu o projeto pro git fazendo com que os demais integrantes façam o fork no git também.

15/04/2026

A professora mandou continuarmos com o a progressão do projeto do TCC abrimos uma nova tag "tecnico de enfermagem/enfermagem, após isso mostramos nosso codigo base para a professora e pedimos um suporte de como poderiamos criar um banco de dados para o site. Ela gostou do codigo que mostramos e indicou o Firebase para a criação do banco de dados.

17/04/2026

Hoje editamos a área paciente.html, adicionamos a opção "telefone 2" como opicional ao paciente. E demos uma pesquisada sobre o banco de dados em SQL e decidimos que faremos no supabase (site que o professor de modelagem está ulitilizando com a gente nas aulas).
